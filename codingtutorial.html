responsive website for for finding and viewing coding tutorials

-By Ayushi Raghave

Project overview

The Product
LearnHub is a website that seeks to provide a high-quality education through coding tutorials that help students develop coding skills and improve their coding knowledge.
The problem:

Most students desire to study coding to further their careers, but they lack the necessary tools to do so. Learnhub offers coding tutorials to assist such students in gaining coding proficiency.

The goal:

Design a responsive website for for finding and viewing coding tutorials

My role:

UX designer leading the responsive website design for coding tutorials

My Responsibilities:

Conducting interviews, paper and digital wireframing, low and high-fidelity prototyping, conducting usability studies, accounting for accessibility, iterating on designs, determining information architecture, and responsive design.

Affinity Diagram

Low-fidelity prototype

To prepare for usability testing, I created a low-fidelity prototype that connected the user flow of how the user can use this coding tutorial website for viewing coding tutorials.

High-fidelity prototype

The high-fidelity prototype followed the same user flow as the low-fidelity prototype, including design changes made after the usability study.

Responsive designs:

The designs for screen size variation included mobile, tablet, and desktop. I optimized the designs to fit specific user needs of each device and screen size.
